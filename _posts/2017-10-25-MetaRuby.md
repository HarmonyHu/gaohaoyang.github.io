### eval函数
该函数将字符串解析成代码，如下示例

* 传入变量字符串，解析成变量
```ruby
test = "abc"
puts eval("test")
#打印abc
puts eval("0x12")
#打印18
```


* 传入函数字符串，解析调用后的结果
```ruby
def func(a)
	return a*a
end
puts eval("func(10)")
#打印100
```
